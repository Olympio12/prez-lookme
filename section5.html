Planning Prévisionnel

Vue d'ensemble

Durée totale : 6 mois (Décembre 2025 - Mai 2026)
Méthodologie : Agile avec sprints de 2 semaines
Jalons clés : 5 phases majeures avec livrables validés

Timeline du Projet

Décembre 2025

Phase 1 : Cadrage & Lancement

Livrables :

  • Cahier des charges validé
  • Charte de projet signée
  • Équipe constituée
  • Environnements techniques préparés
Janvier - Février 2026

Phase 2 : Conception UX/UI & IA

Livrables :

  • Maquettes validées (Figma)
  • Prototype interactif
  • Modèle IA V1 (reconnaissance faciale)
  • Charte graphique finalisée
Mars 2026

Phase 3 : Développement Sprint 1-2

Livrables :

  • Frontend mobile (version alpha)
  • Backend & API fonctionnels
  • Système d'authentification
  • Intégration IA initiée
Avril 2026

Phase 4 : Tests & Ajustements

Livrables :

  • Tests utilisateurs (50 bêta-testeurs)
  • Correctifs bugs critiques
  • Optimisations performances
  • Version pré-production
Mai 2026

Phase 5 : Livraison MVP

Livrables :

  • Application LookMe V1.0
  • Documentation complète
  • Publication App Store & Google Play
  • Plan de déploiement
